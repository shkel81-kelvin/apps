## âš ï¸ Important Notes

1. **PDF Annotations:**
   - Annotations are stored separately, not in PDF file
   - Export annotations to save your work
   - Clearing is permanent (no undo yet)

2. **Excel Preview:**
   - Shows data only, not formulas
   - No cell formatting (colors, borders from original)
   - Download to edit or see formulas

3. **Word Preview:**
   - Basic text and formatting only
   - Images and complex tables may not show
   - Download for full document experience

4. **File Size:**
   - Very large files (>50MB) may be slow
   - Consider downloading large files instead

## ðŸ†˜ Troubleshooting

**PDF not loading?**
- Check file size (very large PDFs may timeout)
- Try downloading and opening locally

**Excel showing "No content"?**
- File may be corrupted
- Try re-uploading
- Download to check in Excel

**Word preview failed?**
- Complex formatting may cause issues
- Use download button to view in Word
- File may be password-protected

**Annotations not showing?**
- Make sure you're on the correct page
- Annotations are per-page
- Check if you accidentally cleared them

**Fullscreen not working?**
- Try clicking minimize then maximize again
- Refresh the page if needed
