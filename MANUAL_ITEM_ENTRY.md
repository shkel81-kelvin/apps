## User Workflow

### Step 1: Prepare
```
1. Open your quotation PDF
2. Keep it visible on screen
```

### Step 2: Enter Items
```
1. Go to "Add Items to Work Order"
2. Click "Add Item" for each item
3. Type:
   - Description
   - Unit (pcs, box, etc.)
   - Quantity
   - Unit Price
4. Amount calculates automatically
```

### Step 3: Select & Add
```
1. Check items you want to add
2. Click "Add X Items to Work Order"
3. Returns to Project Checker
4. Items added!
```
