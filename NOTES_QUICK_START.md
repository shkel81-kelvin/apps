## Best Practices

### For Action Items:
- Always set a due date
- Update status as you progress
- Mark completed when done
- Review pending items weekly

### For Issues:
- Set priority based on impact
- Document steps taken
- Update status when resolved
- Link to related documents

### For Meetings:
- Create note immediately after meeting
- Include key decisions and action items
- Mark completed when minutes finalized
- Reference attendees in content

### For Decisions:
- Document who approved
- Include date decided
- Mark completed immediately
- Link to supporting documents
