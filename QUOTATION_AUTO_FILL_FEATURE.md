## Troubleshooting Auto-Fill

**"Could not auto-detect items"**
- This happens if the PDF is **scanned** (image only).
- Solution: Enter items manually.

**"Descriptions look messy"**
- PDF text layers can sometimes be messy.
- Solution: Edit the description in the table.

**"Wrong prices?"**
- System looks for currency formats.
- Solution: Double-check and correct price in table.
